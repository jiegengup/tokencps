# TokenCPS 隐私政策

**生效日期：2026年4月1日**
**版本号：v2.0**

---

## 第一条 引言

1.1 TokenCPS 平台（以下简称"平台"或"我们"）深知个人信息对您的重要性，并会尽全力保护您的个人信息安全。

1.2 本隐私政策适用于平台提供的所有产品和服务，包括但不限于 Claude API 充值服务、GPT Plus 账号销售服务及推广联盟服务。

1.3 请您在使用平台服务前，仔细阅读并充分理解本隐私政策。一旦您开始使用平台服务，即表示您已阅读、理解并同意本隐私政策。

## 第二条 我们收集的信息

2.1 **账户注册信息**：
- 手机号码（用于账户注册、登录验证及安全通知）
- 电子邮箱（用于账户绑定及服务通知）
- 登录密码（加密存储）

2.2 **身份认证信息**（仅推广员提现时收集）：
- 真实姓名
- 身份证号码
- 支付宝账号（用于佣金提现）

2.3 **支付信息**：
- 支付订单号
- 支付金额及时间
- 支付渠道信息（微信支付/支付宝）

> 注：平台不直接存储用户的银行卡号、支付密码等敏感支付信息，支付过程由第三方支付服务商（虎皮椒支付）处理。

2.4 **服务使用信息**：
- API 调用日志（调用时间、调用模型、Token 用量）
- 账户余额变动记录
- 推广链接点击及转化数据

2.5 **设备与网络信息**：
- IP 地址
- 浏览器类型及版本
- 操作系统信息
- 设备标识符
- 访问时间及页面浏览记录

## 第三条 信息使用目的

3.1 我们收集和使用您的个人信息用于以下目的：

| 使用目的 | 涉及信息 |
|---------|---------|
| 账户管理 | 手机号、邮箱、密码 |
| 服务交付 | 支付信息、API 调用日志 |
| 佣金结算 | 推广数据、身份认证信息、支付宝账号 |
| 风险控制 | IP 地址、设备信息、行为数据 |
| 客户服务 | 手机号、邮箱、服务使用记录 |
| 平台优化 | 设备信息、页面浏览记录（匿名化处理） |

3.2 我们不会将您的个人信息用于上述目的以外的用途。如需变更使用目的，我们将再次征得您的同意。

## 第四条 信息存储

4.1 **存储地点**：您的个人信息存储于中华人民共和国大陆地区的服务器。我们不会将您的个人信息传输至境外。

4.2 **存储期限**：
- 账户信息：自账户注销之日起保留 30 天后删除；
- 交易记录：根据《电子商务法》等法律法规要求，保留不少于 3 年；
- API 调用日志：保留 90 天后自动清理；
- 风控相关数据：保留 1 年。

4.3 **安全措施**：
- 数据传输采用 HTTPS/TLS 加密；
- 敏感信息（密码、身份证号等）采用不可逆加密或脱敏存储；
- 实施严格的数据访问权限控制；
- 定期进行安全审计和漏洞扫描。

## 第五条 信息共享与披露

5.1 我们不会向第三方出售您的个人信息。

5.2 在以下情形下，我们可能会与第三方共享您的部分信息：

| 第三方 | 共享信息 | 共享目的 |
|-------|---------|---------|
| 虎皮椒支付 | 订单金额、支付账户标识 | 支付处理 |
| 灵活用工平台 | 真实姓名、身份证号、银行账户 | 推广员佣金发放及税务合规 |

5.3 在以下法定情形下，我们可能会披露您的个人信息：
- （a）经您本人明确同意；
- （b）根据法律法规的要求；
- （c）根据政府主管部门的强制性要求；
- （d）为维护平台及其他用户的合法权益。

## 第六条 您的权利

6.1 **查询权**：您有权查询我们持有的您的个人信息。您可通过平台账户设置页面查看。

6.2 **更正权**：当您发现我们持有的您的个人信息有误时，您有权要求我们更正。

6.3 **删除权**：在以下情形下，您有权要求我们删除您的个人信息：
- （a）处理目的已实现或无法实现；
- （b）我们违反法律法规或与您的约定收集、使用个人信息；
- （c）您注销账户。

6.4 **注销权**：您有权注销平台账户。注销后，我们将在合理期限内删除或匿名化处理您的个人信息，法律法规另有规定的除外。

6.5 **撤回同意权**：您有权撤回此前授予的同意。撤回同意不影响撤回前基于同意所进行的信息处理。

## 第七条 未成年人保护

7.1 平台服务面向年满 18 周岁的成年人。我们不会故意收集未满 18 周岁未成年人的个人信息。

7.2 如果我们发现在未获得可验证的监护人同意的情况下收集了未成年人的个人信息，我们将尽快删除相关信息。

## 第八条 Cookie 及类似技术

8.1 平台使用 Cookie 及类似技术来提供、保护和改进服务，包括记住您的登录状态、分析平台使用情况等。

8.2 您可以通过浏览器设置管理 Cookie。但请注意，禁用 Cookie 可能影响平台部分功能的正常使用。

## 第九条 隐私政策的变更

9.1 我们可能会不时修订本隐私政策。修订后的隐私政策将在平台公示，重大变更将通过站内通知等方式告知您。

9.2 修订后的隐私政策自公示之日起生效。如您在隐私政策修订后继续使用平台服务，即视为您同意修订后的隐私政策。

## 第十条 联系我们

10.1 如您对本隐私政策有任何疑问、意见或建议，可通过以下方式联系我们：
- 平台内反馈入口
- 客服微信/企业微信

10.2 我们将在收到您的请求后 15 个工作日内予以回复。
