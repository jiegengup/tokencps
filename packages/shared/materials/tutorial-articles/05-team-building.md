# 团队裂变：邀请下级推广员

> 一个人推广有天花板，团队裂变才能指数增长。本文教你如何通过邀请下级推广员，建立自动化的推广团队。

---

## 一、团队裂变机制

### TokenCPS 的二级分销模型

```
你（上级推广员）
├── 你自己推广 → 佣金 50%（无上级）或 40%（有上级）
├── 下级推广员 A → 他的每笔成交，你抽 10%
├── 下级推广员 B → 他的每笔成交，你抽 10%
└── 下级推广员 C → 他的每笔成交，你抽 10%
```

### 收益计算

| 来源 | 计算方式 | 示例 |
|------|---------|------|
| 自己推广 | 用户充值 × 50% | 用户充 ¥200 → 你拿 ¥100 |
| 下级推广 | 下级用户充值 × 10% | 下级用户充 ¥200 → 你拿 ¥20 |
| 邀请奖励 | 每邀请 1 人 | ¥5（从下级首笔佣金扣） |

关键点：下级推广员的分成是**永久**的，只要他还在推广，你就持续有收入。

## 二、去哪里找推广员？

### 1. 技术圈

- **独立开发者**：有自己的产品和用户群
- **技术博主**：有内容创作能力和粉丝
- **开源项目作者**：有 GitHub 流量
- **技术社群群主**：有精准的技术用户群

### 2. 自媒体圈

- **AI 领域自媒体**：公众号、视频号、小红书
- **科技评测博主**：有 AI 工具评测需求
- **知识付费从业者**：有付费用户群体

### 3. 副业圈

- **副业交流群**：寻找新项目的人
- **站长/SEO 从业者**：有流量变现需求
- **社群运营者**：有社群资源

## 三、招募话术

### 话术 1：直接招募

> "在做一个 AI 领域的推广联盟，佣金 50%，用户充 ¥200 你拿 ¥100。产品是 Claude API 充值，技术圈刚需。你有 [XX渠道]，很适合做这个。感兴趣的话我发你注册链接？"

### 话术 2：收益驱动

> "我最近在推广 Claude API 充值，一个月被动收入 ¥XXXX（展示截图）。佣金 50%，用户续充还持续有佣金。你要不要也试试？我可以教你怎么做。"

### 话术 3：产品驱动

> "你平时用 Claude API 吗？TokenCPS 这个充值渠道不错，人民币直充。他们有推广联盟，佣金 50%，你本来就在用，顺手推荐给朋友就能赚钱。"

> 更多话术参考：[社群话术模板](../community-scripts.md)

## 四、如何帮助下级推广员成功

你的下级推广员赚得越多，你的 10% 分成就越多。所以帮助他们成功就是帮助自己。

### 1. 提供入门指导

- 分享本系列教程给下级推广员
- 教他们如何获取推广链接
- 帮他们选择适合的推广渠道

### 2. 分享推广素材

- 转发朋友圈文案模板
- 分享社群话术
- 提供 GitHub README 模板

### 3. 分享经验

- 告诉他们哪些渠道效果好
- 分享你的转化率数据
- 教他们避免常见错误

### 4. 建立推广员社群

- 创建微信群，把你的下级推广员拉进来
- 定期分享推广技巧和平台动态
- 互相交流经验，共同成长

## 五、团队规模与收入

### 收入模型

假设你邀请了 20 个下级推广员，每人每月带来 ¥5,000 成交：

| 收入来源 | 计算 | 月收入 |
|---------|------|-------|
| 自己推广（假设 ¥10,000 成交） | ¥10,000 × 50% | ¥5,000 |
| 20 个下级推广员 | 20 × ¥5,000 × 10% | ¥10,000 |
| **合计** | | **¥15,000** |

其中 ¥10,000 来自团队，是真正的被动收入。

### 团队增长路径

| 阶段 | 团队规模 | 月被动收入（团队部分） |
|------|---------|-------------------|
| 起步 | 3-5 人 | ¥1,500-2,500 |
| 成长 | 10-20 人 | ¥5,000-10,000 |
| 成熟 | 50+ 人 | ¥25,000+ |

## 六、团队管理技巧

### 1. 设定预期

- 告诉下级推广员真实的收入预期
- 不要过度承诺，避免后续纠纷
- 强调这是长期项目，不是快钱

### 2. 保持沟通

- 每周在推广员群里分享一次平台动态
- 及时回答下级推广员的问题
- 分享成功案例激励团队

### 3. 淘汰与激励

- 对活跃推广员给予更多支持和指导
- 对不活跃的推广员不必强求
- 持续招募新推广员，保持团队活力

---

## 七、注意事项

1. **合规推广**：确保下级推广员也遵守平台规则，不刷单、不虚假推广
2. **如实介绍**：招募时如实说明佣金规则和提现条件
3. **不要传销化**：TokenCPS 是二级分销（你 + 下级），不是多级传销
4. **关注质量**：10 个活跃推广员比 100 个僵尸推广员有价值得多

---

## 总结

团队裂变的核心逻辑：

1. **找到对的人**：有流量、有渠道、有推广意愿的人
2. **帮助他们成功**：提供素材、教程、经验
3. **持续扩大团队**：不断招募新推广员
4. **享受被动收入**：团队越大，被动收入越多

AI 行业的增长红利还在，现在建立团队就是在抢占先机。
