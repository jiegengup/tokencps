# 新手入门：3 分钟开始推广

> 本文面向刚注册 TokenCPS 的新推广员，手把手教你从 0 到 1 开始推广赚钱。

---

## 什么是 TokenCPS？

TokenCPS 是一个 AI 领域的 CPS 联盟平台。你通过分享推广链接，引导用户购买 Claude API 充值或 GPT Plus 账号，就能获得高达 50% 的佣金分成。

简单说：**分享链接 → 用户购买 → 你拿佣金**。

---

## 第一步：注册推广员账号

1. 打开 TokenCPS 推广员注册页面
2. 填写手机号/邮箱，设置密码
3. 完成注册，进入推广员后台

> 注册完成后，你就是 TokenCPS 推广员了。零门槛，不需要审核。

---

## 第二步：获取你的专属推广链接

1. 登录推广员后台
2. 进入「推广链接」页面
3. 系统已为你自动生成专属推广链接
4. 点击「复制链接」

你的推广链接格式类似：`https://tokencps.com/p/你的推广码`

用户通过这个链接注册并购买，佣金就归你。

---

## 第三步：分享推广链接

把你的推广链接分享出去就行。常见的分享渠道：

| 渠道 | 适合人群 | 效果 |
|------|---------|------|
| 微信朋友圈 | 所有人 | 覆盖面广 |
| 技术群/微信群 | 开发者 | 转化率高 |
| GitHub README | 开源项目作者 | 精准流量 |
| 技术博客 | 内容创作者 | 长尾流量 |
| 社交媒体 | KOL/自媒体 | 传播快 |

> 平台还提供推广海报一键生成功能，自动带你的二维码，方便在微信等场景分享。

---

## 佣金怎么算？

| 场景 | 你的佣金 |
|------|---------|
| 用户充值 ¥100（无上级） | ¥50（50%） |
| 用户充值 ¥200（无上级） | ¥100（50%） |
| 用户充值 ¥200（有上级） | ¥80（40%），上级拿 ¥20（10%） |
| 用户续充 ¥100 | 继续拿佣金 |

- Claude API：按用户实际使用量最终结算
- GPT Plus：购买即确认，一次性结算（70% 佣金）

---

## 佣金怎么提现？

1. 完成实名认证（提现时才需要）
2. 佣金 T+7 后可提现（订单完成 7 天后）
3. 最低 ¥1 起提
4. 提现到支付宝账户

---

## 进阶：邀请下级推广员

你还可以邀请其他人成为推广员：

- 每成功邀请一个推广员 → 奖励 ¥5
- 下级推广员的每笔成交 → 你永久抽 10%

这意味着你不仅自己推广赚钱，还能通过团队裂变获得被动收入。

## 常见问题

**Q：需要花钱才能开始吗？**
A：不需要。注册免费，推广免费，零成本开始。

**Q：用户退款了佣金怎么办？**
A：用户退款时，对应佣金会同步追回。只有用户实际使用的部分才算佣金。

**Q：我自己买也有佣金吗？**
A：有。通过自己的推广链接购买，同样享受佣金（自买省钱）。

**Q：推广链接有效期多久？**
A：永久有效。用户最后点击的推广链接决定佣金归属。
