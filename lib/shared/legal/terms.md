## 第一条 总则

1.1 本协议（以下简称"本协议"）是您（以下简称"用户"）与 TokenCPS 平台（以下简称"平台"或"我们"）之间就使用平台服务所订立的有效合约。

1.2 平台域名为 tokencps.com，提供 Claude API 充值服务及 GPT Plus 账号销售服务（以下统称"平台服务"）。

1.3 用户在注册、登录或使用平台服务前，应当仔细阅读并充分理解本协议全部条款。一旦用户注册、登录或以其他方式使用平台服务，即视为用户已阅读、理解并同意接受本协议的约束。

## 第二条 服务内容

2.1 **Claude API 充值服务**：用户通过平台以人民币充值，获取 Claude API 使用额度（以美元计价）。平台自动发放 API Key 及接入地址，用户可按照平台提供的配置教程接入使用。

2.2 **GPT Plus 账号销售服务**：平台向用户提供 GPT Plus 共享账号或独享账号，以账号密码形式交付。

2.3 **推广联盟服务**：用户可申请成为推广员，通过分享推广链接获取佣金收入。推广员服务的具体规则详见《推广员合作协议》。

2.4 平台保留随时新增、变更或终止部分服务内容的权利，并将提前在平台公告。

## 第三条 用户注册与账户

3.1 用户应以真实身份信息注册平台账户，包括但不限于手机号码、电子邮箱等。

3.2 用户应妥善保管账户信息及密码，因用户保管不善导致的损失由用户自行承担。

3.3 每位用户仅可注册一个平台账户。平台有权对异常注册行为进行限制或封禁处理。

3.4 用户账户下可创建多个 API Key，所有 Key 共享账户余额。

## 第四条 充值与支付

4.1 用户通过平台支持的支付方式（微信支付、支付宝等）进行充值，充值金额以人民币计价，账户余额以美元额度显示。

4.2 充值成功后，平台将即时为用户账户增加对应额度。具体汇率及额度换算以平台实时显示为准。

4.3 用户应确保支付账户为本人所有，因使用他人支付账户引发的纠纷由用户自行承担。

4.4 充值订单一经支付成功，不可撤销。退款事宜按照本协议第五条执行。

## 第五条 退款规则

5.1 **Claude API 充值退款**：
- （a）用户可申请退款，退款金额按账户剩余未使用额度计算；
- （b）已使用的额度不予退还；
- （c）退款将原路返回至用户支付账户，处理周期为 3-7 个工作日；
- （d）退款时，与该订单关联的推广员佣金将同步追回，上级推广员分成亦同步追回；
- （e）因退款导致推广员账户余额为负的，平台有权从推广员后续佣金中扣除。

5.2 **GPT Plus 账号退款**：
- （a）GPT Plus 账号属于虚拟商品，一经交付（发放账号密码），不支持退款；
- （b）因账号本身质量问题（如无法登录）导致无法使用的，用户应在交付后 24 小时内联系平台客服，平台将视情况予以换号处理；
- （c）因用户自身操作不当（如修改密码、违规使用导致封号等）造成的账号不可用，平台不承担任何责任，不予退款或换号。

5.3 平台保留对异常退款行为（如频繁充值后立即退款）进行风控限制的权利。

## 第六条 GPT Plus 账号特别条款

6.1 GPT Plus 账号由第三方供应商提供，平台仅作为销售渠道。

6.2 **无质量保证**：平台不对 GPT Plus 账号的持续可用性、稳定性、功能完整性作任何明示或暗示的保证。

6.3 **账号风险**：GPT Plus 账号可能因第三方平台政策变更、账号共享检测等原因被限制或封禁，此类风险由用户自行承担。

6.4 **使用限制**：用户不得将 GPT Plus 账号转售、共享给第三方，不得利用账号从事违法违规活动。

- （c）用户自身网络环境问题；
