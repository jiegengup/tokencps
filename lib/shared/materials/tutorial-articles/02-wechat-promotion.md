# 微信推广实战指南

> 微信是国内最大的社交平台，也是 TokenCPS 推广转化率最高的渠道之一。本文教你如何在微信生态中高效推广。

---

## 一、朋友圈推广

### 发布策略

- **频率**：每天 1-2 条，不要刷屏
- **时间**：早 8-9 点（通勤）、中午 12-13 点（午休）、晚 20-22 点（休闲）
- **内容**：产品介绍、使用体验、收益截图交替发布

### 文案技巧

1. **开头抓眼球**：用数据或痛点开头，比如"Claude API 人民币直充，不用再折腾信用卡了"
2. **突出利益点**：价格优势、操作简单、新人福利
3. **加入社交证明**：已有 XXX 人使用、佣金收入截图（脱敏）
4. **明确行动指引**：文末放推广链接或二维码

### 配图建议

- 平台界面截图（展示产品真实性）
- 佣金收入截图（脱敏处理，展示赚钱效果）
- 推广海报（平台一键生成，带你的二维码）

> 参考文案模板：查看 [朋友圈文案模板](../wechat-moments.md)

## 二、微信群推广

### 选择目标群

| 群类型 | 推广方向 | 转化率 |
|-------|---------|-------|
| 技术交流群 | Claude API 充值 | 高 |
| AI 学习群 | Claude API + GPT Plus | 高 |
| 副业/赚钱群 | 推广员招募 | 中 |
| 泛兴趣群 | GPT Plus 账号 | 低 |

### 推广技巧

1. **先融入再推广**：先在群里正常交流，建立信任后再推荐
2. **解决问题式推广**：当有人问 AI 相关问题时，自然地推荐
3. **不要硬广**：避免直接丢链接，用分享体验的方式
4. **提供价值**：分享 AI 使用技巧，顺带推荐充值渠道

> 参考话术：查看 [社群话术模板](../community-scripts.md)

## 三、私聊推广

### 适用场景

- 朋友问你 AI 相关问题时
- 看到朋友发 AI 相关朋友圈时
- 有人主动咨询时

### 私聊话术框架

1. **共情**：理解对方的需求或痛点
2. **推荐**：自然地推荐 TokenCPS
3. **说明**：简要说明优势（2-3 个点）
4. **引导**：给出推广链接

示例：
> "你说想用 Claude API 但充值麻烦？我一直用 TokenCPS，人民币直充，微信支付就行。自动发 Key，还有配置教程。新用户送 $5 体验额度，你可以先试试：{{推广链接}}"

## 四、微信公众号/视频号

### 公众号推广

- 写 AI 相关教程文章，文末放推广链接
- 发布 Claude/GPT 使用技巧，自然植入
- 做 AI 工具评测，推荐充值渠道

### 视频号推广

- 录制 AI 工具使用教程
- 展示 Claude API 接入过程
- 分享推广收益（激励观众注册）

---

## 五、避坑指南

1. **不要刷屏**：朋友圈每天最多 2 条，群里不要连续发
2. **不要夸大**：如实介绍产品，不要承诺"稳赚"
3. **不要群发**：微信群发容易被举报，影响账号
4. **注意合规**：不要在推广中使用"最便宜""保证"等绝对化用语
5. **尊重群规**：有些群禁止广告，不要硬推

---

## 六、数据复盘

每周检查推广数据：

- **点击量**：推广链接被点击了多少次
- **注册量**：有多少人通过你的链接注册
- **转化率**：注册用户中有多少人充值
- **佣金收入**：本周赚了多少

根据数据调整策略：
- 点击多但注册少 → 优化落地页引导
- 注册多但充值少 → 加强产品价值传达
- 某个渠道效果好 → 加大该渠道投入
