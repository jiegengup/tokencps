# TokenCPS 产品开发指南

> 写给非技术背景的创始人：你的产品是什么、怎么运作、现在做到哪了、下一步该做什么。

## 📌 产品一句话

**你做的就是一个"AI 模型的淘宝联盟"** — 别人通过你的平台买 Claude/GPT 的 API，推广员帮你拉客户赚佣金，你赚中间差价。

## 🔄 产品怎么赚钱（核心商业模型）

```
用户充 ¥100 → 到账 $100 API 额度
     ↓
用户用 Claude 消耗 $100
     ↓
你的成本：¥20（selao货源）或 ¥35（xinxu货源）
     ↓
推广员佣金：¥50（50%，按消耗结算）
     ↓
你的利润：¥100 - ¥20 - ¥50 = ¥30
```

**关键：用户充的钱不是立刻花掉的，是按实际 API 调用消耗的。推广员佣金按消耗结算，不是按充值。**

| 角色 | 是谁 | 干什么 |

## 🎯 下一步该做什么（建议路线）

### 第一步：货源对接（最关键）
**目标：** 用户充值后能真正拿到 Claude API Key
**怎么做：**
1. 对接 selao 中转站，充值时调用它的 API 创建 Key
2. 把 Key 返回给用户，用户直接用
3. 用户消耗的金额回传给 selao 做结算

### 第二步：注册送 $5
**目标：** 降低用户门槛，注册就能体验
**怎么做：**
1. 注册成功后自动给账户加 $5 余额
2. 这 $5 只能买 Key 用，不能提现
3. 推广员拉新也有 $5 的预估佣金

### 第三步：成长体系
**目标：** 让推广员有动力长期推广
**怎么做：**
| 累计佣金 | 档位 | 佣金比例 |
|----------|------|---------|
| < ¥1000 | 铜牌 | 30% |
| ¥1000-4999 | 银牌 | 40% |
| ¥5000-19999 | 金牌 | 50% |
| ≥ ¥20000 | 钻石 | 60% |

## 🔑 你需要记住的关键数字

| 数字 | 值 | 说明 |
|------|---|------|
| 用户充值 | ¥1 = $1 | 1:1 到账 |
| 货源成本 | ¥0.20-$1 | selao 高级代理 |
| 最大佣金 | 50% | 按消耗结算 |
| 二级佣金 | 10% | 上级抽成 |
