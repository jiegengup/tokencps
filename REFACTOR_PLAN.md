# TokenCPS v2.0 重构计划

## 架构决策
- **Monorepo 结构**：用 Next.js 的路由组实现两套前端（不拆包，共享后端）
  - `/promoter/*` — 推广员端（移动端优先）
  - `/` — C端购买平台（PC端优先）  
  - `/admin/*` — 管理后台
  - `/api/*` — 统一后端 API
- **UI 风格**：claude.ai 风格（米白背景、棕色强调、大留白、圆角）
- **可复用**：JWT中间件、TypeORM实体骨架、Zustand store 模式
